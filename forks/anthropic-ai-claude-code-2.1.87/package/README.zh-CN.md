# Claude Code

![](https://img.shields.io/badge/Node.js-18%2B-brightgreen?style=flat-square) [![npm]](https://www.npmjs.com/package/@anthropic-ai/claude-code)

[npm]: https://img.shields.io/npm/v/@anthropic-ai/claude-code.svg?style=flat-square

Claude Code 是一款运行在终端中的智能编程工具：它能理解你的代码库，通过自然语言指令帮你更快写代码——执行日常任务、讲解复杂代码、处理 Git 工作流等。可在终端、IDE 中使用，或在 GitHub 上 @claude。

**更多介绍见 [Claude Code 官网](https://claude.com/product/claude-code)** | [文档](https://code.claude.com/docs/en/overview)

<img src="https://github.com/anthropics/claude-code/blob/main/demo.gif?raw=1" />

## 快速开始

1. 安装 Claude Code：

```sh
npm install -g @anthropic-ai/claude-code
```

1. 进入项目目录并运行 `claude`。

## 反馈问题

欢迎你的反馈。可在 Claude Code 内使用 `/bug` 命令直接上报问题，或在 [GitHub Issues](https://github.com/anthropics/claude-code/issues) 提交工单。

## Discord 社区

加入 [Claude 开发者 Discord](https://anthropic.com/discord)，与其他使用 Claude Code 的开发者交流：获取帮助、分享反馈、讨论项目。

## 数据收集、使用与保留

使用 Claude Code 时，我们会收集反馈信息，其中包括使用数据（例如代码采纳或拒绝情况）、相关对话数据，以及通过 `/bug` 命令提交的用户反馈。

### 我们如何使用你的数据

请参阅我们的 [数据使用政策](https://code.claude.com/docs/en/data-usage)。
